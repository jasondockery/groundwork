# Code of Conduct

This project is meant to help people learn. Be clear, kind, and practical.

Expected behavior:

- Assume good intent and explain tradeoffs honestly.
- Make room for beginners while keeping technical standards high.
- Keep feedback specific to the work.
- Do not post private information, secrets, work data, or personal notes.

Unacceptable behavior:

- Harassment, threats, or personal attacks.
- Publishing someone else's private information.
- Pressuring people to run scripts they do not understand.

The repository owner may remove comments, close issues, or block participants
who make the project less safe or useful for learners.
