# raw

Drop source material here: articles, PDFs, transcripts, clippings, notes,
screenshots, and supporting assets.

These files are immutable: the agent only reads them, never edits or deletes
them. After adding something, tell the agent to ingest it.

Suggested folders:

- `articles/`
- `papers/`
- `meetings/`
- `screenshots/`
- `assets/`
- `clippings/`
- `notes/`
- `transcripts/`
