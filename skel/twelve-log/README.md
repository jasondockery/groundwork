# Groundwork Twelve — learning log

This repo is your progress artifact for [Groundwork Twelve](https://jasondockery.github.io/groundwork/groundwork-twelve.html):
the daily log, weekly reviews, and every Saturday build live here. By Day 84
it is a portfolio of proof, not a certificate.

## Use

1. One line in `log.md` per day — what you did, how it felt, minutes spent.
2. Commit every day, even minimum days. The streak is `git log`.
3. Each Sunday, add a short weekly review under a heading: what got easier,
   what still feels slow, one thing to repeat.
4. Saturday builds get committed here too (or linked, once they grow into
   their own repos).
5. From Week 4 on, run the examiner once a week: start your agent here and
   paste `prompts/examiner.md`.

Push this repo to GitHub if public accountability helps you — many people
finish because someone might be watching.

## Day modes

- **Full day:** 60–75 minutes, the plan as written.
- **Minimum day:** 15 minutes — one drill, one log line, one commit. Real day.
- **Catch-up day:** 90–120 minutes. Never stack more than one missed day.
